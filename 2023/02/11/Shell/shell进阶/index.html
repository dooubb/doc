获取当前所执行的shell的文件夹

${BASH_SOURCE[0]}表示bash脚本的第一个参数（如果第一个参数是bash，表明这是要执行bash脚本，这时"${BASH_SOURCE[0]}"自动转换为第二个参数），例如：

"${BASH_SOURCE[0]}"代表的是/data/shell/shell.sh。
"dirname"表示提取参数里的目录，dirname "${BASH_SOURCE[0]}"表示提取bash脚本第一个参数里的目录，例如/data/shell/shell.sh”的目录为"/data/shell。
cd "$( dirname "${BASH_SOURCE[0]}" )“表示切换到刚才提取的目录，
dir=cd "$( dirname "${BASH_SOURCE[0]}" )" && pwd则表示，如果第一条语句顺利执行，就执行pwd显示当前目录，并将结果赋值给变量dir。
cd "${dir}/..切换到dir变量所指目录的上一级目录。
